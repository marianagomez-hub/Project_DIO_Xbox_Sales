# Project_DIO_Xbox_Sales
Foi criado um dashboard a partir de uma base da dados contendo diferentes informações de assinantes de Xbox, como renovação automatica, tipo de plano e assinatura, se possui renovação automatica ou não, entre outros dados.
No nosso dashboard temos, à esquerda, um painel que permite escolher entre o tipo de assinatura: mensal, anual e trimestral. À direita temos dois grandes numeros, o primeiro representando o total de assinaturas do EA PLAY SEASON PASS e o segundo do total de assinaturas do Minecraft Season Pass
Abaixo, temos um grafico representando o total de assinaturas do Xbox Game Pass, divido entre aquelas que tem renovação automatica (YES) e aquelas que não possuim a renovação ativada (NO)
No painel da direita podemos escolher o tipo de assinatura, que vai modificar os nossos 3 resultados.
